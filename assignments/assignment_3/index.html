Omschrijving

Het doel van deze opdracht is om deze website te splitsen naar een separate HTML en CSS file.

  1. Sla deze pagina eerst op als bestand door op de rechter muisknop te klikken en vervolgens op opslaan.
  2. Maak een map aan met de naam "opdracht_3".
  3. Maak in Notepad++ twee bestanden aan, "index.html" en "styles.css". Deze sla je op in de map.
  4. Kopieer de elementen uit dit bestand naar het juiste bestand en link het CSS bestand aan het HTML bestand.

Prachtig CSS stylesheet.
