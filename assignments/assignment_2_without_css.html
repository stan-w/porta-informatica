Omschrijving

Voor deze opdracht moet je weer een website namaken. Ditmaal hoef je niet zelf de HTML code te schrijven, maar alleen de CSS.

Voeg de CSS "internal" toe. Dit betekent bovenin de code als style element. Er is al een begin gemaakt waar je verder in kan werken. De lay-out moet je regelen met floats.
